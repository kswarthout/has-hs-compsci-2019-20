# Create a Python class to represent an HAS course

### Defining The Class
- The class should have the properties title, classroom, description, and students
- The class should contain a method add_student(student) that accepts a string to represent a student name, and it should store the name in the students list.

### Testing The Class
- Create a course instance, initializing each property inside the constructor
- Call the method add_student on the course instance. 
- Print the course details to the console by displaying each of the properties

### Sample Output:

```
Title: HSCS 2019-2020  
Room: Computer Lab  
Desc: High School Computer Science  
Students: ['Chris Tung']
```
